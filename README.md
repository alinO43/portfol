# Портфолио · Алина Тотоева

Personal portfolio website showcasing UX/UI design projects and experience.

## 📋 About

UX/UI designer portfolio featuring:
- **4 case studies** with detailed design processes
- **Responsive design** optimized for all devices
- **Modern tech stack** with clean HTML/CSS/JS
- **Static site** for fast performance and easy deployment

## 🚀 Quick Start

### View Locally
```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (npx serve)
npx serve .

# Then open http://localhost:8000
```

### Deploy

**GitHub Pages:**
```bash
# Push to main branch
git push origin main
# Enable GitHub Pages in repository settings
# Select "Deploy from a branch" → main
```

**Vercel:**
```bash
npm i -g vercel
vercel
```

## 🎨 Design System

- **Colors**: Dark theme with blue accent (#6C7BFF)
- **Typography**: Archivo (headings), Hanken Grotesk (body)
- **Spacing**: Consistent grid with 14px-84px spacing
- **Components**: Cards, badges, pills, galleries, forms
- **Responsive**: Mobile-first, tested on 480px+ devices
